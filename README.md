<h2>AluraTube</h2>

<p>Clone of Youtube where it is possible to list videos by categories. It is also possible to choose between light or dark tone.</p>

<h4>Technologies used</h4>
<ul>
  <li>NextJS</li>
  <li>React Hooks</li>
  <li>Componentization</li>
  <li>Styled Components</li>
</ul>
